 # Coffee Personality Quiz ☕

  Интерактивный квиз "Какая ты кофейная личность?" от кофейни «Берёзка».

  ## Результаты

  - **Эспрессо** ☕ — быстрый и энергичный
  - **Латте** 🤍 — мягкий и уютный
  - **Капучино** ☁️ — контрастный и фокусированный
  - **Американо** 🔌 — простой и классический

  ## Запуск локально

  \```bash
  git clone https://github.com/zanasai/quiz-project.git
  cd quiz-project
  npm install
  npm run dev
  \```

  ## Технологии

  Next.js · React · TypeScript · Tailwind CSS
